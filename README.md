# SWR_Online
A tool to connect users and collect SWEP1R Data to send

## Requirements
- QMake
- ASIO

## Compiling
```Shell
git clone --recurse-submodules https://github.com/goofbrush/SWR_Online.git
```
### [Linux] Using QMake from terminal
```
cd SWR_Online
qmake SWR_Client.pro
make
./SWR_Client
```
